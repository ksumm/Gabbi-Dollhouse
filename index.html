Gabby's Dollhouse Memory Game

score: 0

How to play:

  • 1. Begin the game by clicking any card to flip it over.
  • 2. Flip a second card to see if you have a match. Only two cards can be flipped at a time.
  • 3. If you have a match, the cards will remain face up.
  • 4. Find all 8 pairs to win the game!
